### Test case for the checkout option ###

### TC-006:Checkout ###

**Preconditions:**

User is on the cart page

User has at least one product in the cart 

**Steps:**

1.Navigate to the cart page

2.Click the "Checkout" button

**Expected result: User is redirected to the checkout page successfully**

**Actual result: The user is not redirected, the page only refreshes**
